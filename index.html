How To Use

  • Click the "+" icon to create a new box.
  • Click the "remove" button to delete the last box.
  • Click the "Clear Data" button to erase all current Storyline Notepad data (theme not included).
  • Click the title to modify it.
  • You can change the theme using the "Change Theme" dropdown menu.
  • Saving is automatic; it happens when you leave the page.
  • Hint: Hold Shift while using the mouse wheel to scroll horizontally.
